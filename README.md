# 📚 StudyHub

Приложение для учёбы — ведёшь предметы, задания, оценки, расписание, конспекты и расходы в одном месте.

Работает как **десктопное приложение** (Windows, offline-first, SQLite) и как **веб-версия** (Supabase, кросс-платформенно).

---

## ✨ Возможности

### Предметы и задания
- Создавай предметы с цветными метками и привязкой к семестру
- Добавляй задания со статусом, приоритетом и дедлайном
- Подзадачи (чеклист) внутри каждого задания с полоской прогресса
- Повторяющиеся задания (ежедневно / еженедельно / свой интервал)
- Теги для группировки заданий между предметами
- Поиск и фильтры по статусу, приоритету, тегу, дедлайну

### Файлы
- Прикрепляй файлы к заданиям — они копируются в папку приложения
- Можно прикрепить сразу несколько файлов или целую папку с деревом файлов
- Дубликаты определяются автоматически и пропускаются
- Можно удалить оригинал с компьютера, копия останется
- Превью изображений и PDF прямо в приложении
- Выгрузка файлов по фильтру размера одной кнопкой

### Оценки
- Учёт оценок по каждому предмету со средневзвешенным баллом
- Настраиваемая шкала (5, 10, 100 баллов и т.д.)
- Статистика: гистограмма оценок, медиана, мода
- Сводная таблица по всем предметам на дашборде
- Сортировка предметов по среднему баллу

### Расписание и время
- Расписание занятий: вид «Неделя» (таблица слотов × дни) и вид «Месяц» (календарная сетка)
- Цветовая кодировка типов занятий: Лекции → синий, Лабораторные → оранжевый, Практические → зелёный
- Импорт расписания из ТулГУ (tulsu.ru) по номеру группы с предпросмотром изменений
- Автосинхронизация расписания ТулГУ: интервал 3 ч / 6 ч / 12 ч / 24 ч или вручную
- Уведомление об изменениях в расписании (добавлено / удалено / перенесено)
- Календарь с дедлайнами
- Помодоро-таймер с настраиваемыми интервалами
- Учёт времени по предметам и заданиям

### Заметки и конспекты
- Markdown-редактор с автосохранением
- Конспекты привязаны к предмету
- Поиск по заголовку и тексту

### Кошелёк
- Учёт доходов и расходов с категориями (иконка + цвет)
- Фильтр по периоду: месяц, 3 месяца, год, всё
- Статистика: баланс, SVG-диаграммы по дням и категориям
- Экспорт в CSV одной кнопкой
- 13 категорий по умолчанию (Еда, Транспорт, Стипендия и др.)

### Moodle ТулГУ
- Авторизация через Moodle-аккаунт ТулГУ
- Загрузка курсов и привязка к предметам StudyHub
- Скачивание заданий и файлов из Moodle прямо в приложение
- Автосинхронизация раз в сутки при запуске

### Организация
- Семестры: привязывай предметы, смотри архив по семестрам
- Архив завершённых предметов
- Сортировка предметов: по алфавиту, по семестру, по баллу

### Дашборд
- Прогресс по заданиям, ближайшие дедлайны, учебное время
- Серия дней учёбы (streak)
- Текущий средний балл

## ☁️ Облачная синхронизация (опционально)

Подключи Supabase — данные будут синхронизироваться между всеми устройствами.

**Как подключить (десктоп):**
1. Создай проект на [supabase.com](https://supabase.com) (бесплатный план)
2. Открой **Настройки → Supabase** и введи `Project URL` и `anon key`
3. Данные начнут синхронизироваться автоматически

**Роли пользователей:**
- `student` — стандартная роль (по умолчанию)
- `admin` — доступ к панели администратора (список пользователей, смена ролей) и разделу «Кошелёк»

Для активации ролей примени SQL-скрипты из папки `supabase/`:
```
supabase/profiles.sql   — таблица профилей и RLS-политики
supabase/wallet.sql     — таблицы кошелька и RLS-политики
```

## 🌐 Веб-версия

Веб-приложение доступно по адресу, настроенному в Vercel.

Все данные хранятся в Supabase — требуется аккаунт и настроенный проект.

**Запуск веб-версии локально:**
```bash
cd web
cp .env.example .env          # заполни VITE_SUPABASE_URL и VITE_SUPABASE_ANON_KEY
npm install
npm run dev
```

**Сборка для деплоя:**
```bash
npm run build
```

Готовые файлы появятся в `web/dist/`.

## 🖥️ Десктопная версия

### Требования

- Windows 10 / 11
- Node.js 18+ (только для запуска из исходников)

### Готовый .exe (рекомендуется)

Скачай последний релиз со страницы [GitHub Releases](https://github.com/MrSody71/StudyHub/releases):

| Файл | Описание |
|---|---|
| `StudyHub-Setup-*.exe` | Установщик NSIS — ярлык на рабочем столе, автообновления |
| `StudyHub-Portable-*.exe` | Один файл, без установки, можно с флешки |

### Автообновления

Установленная версия тихо проверяет наличие обновлений через 5 секунд после запуска.
Если выходит новая версия — появляется диалог «Доступна новая версия X.X.X. Обновить сейчас?»
с кнопками **Обновить** и **Позже**.

Проверить обновление вручную: **Настройки → Проверить обновления**.

### Из исходников

```bash
git clone https://github.com/MrSody71/StudyHub
cd StudyHub
npm install
npm start
```

### Сборка .exe

```bash
npm run build:win        # установщик (NSIS Setup.exe)
npm run build:portable   # портативная версия (один .exe)
```

Готовые файлы появятся в папке `dist/`.

## 📁 Где хранятся данные (десктоп)

```
C:\Users\ИМЯ\AppData\Roaming\StudyHub\
  studyhub.db      — база данных SQLite
  attachments\     — прикреплённые файлы
```

Данные не удаляются при переустановке приложения.

### Бэкап и перенос

Все данные хранятся в одной папке — просто скопируй её целиком.

**Создать бэкап:** скопируй папку `StudyHub` в безопасное место.

**Восстановить:** замени папку `StudyHub` в `AppData\Roaming` сохранённой копией.

**Перенос на другой компьютер:** скопируй папку `StudyHub` на новый компьютер по тому же пути, затем установи приложение.
